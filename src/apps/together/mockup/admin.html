What is the name of your app?
CodeSync

What collaborative activity can people do together using your app?
People can code, compile, and execute code together.

What is the ideal number of concurrent users (must be at least 10 but no more than 40)?
The ideal number is 15 since more than that can become counter productive.

What skills do people need in order to do this activity?
Coding skills.

Is this activity for fun or for something practical? This activity is meant to increase productivity.

How can a person see what everyone else is doing?
The code is displayed dyanmically so everyone is seeing the code in realtime.

How can a person see the most recent result of everyone’s collaborative effort?
By looking at the "Code" tab or checking the recent changes log.

What can an admin see (i.e., God’s mode)?
Administrators have access to user permissions, which means that they can invite/uninvite users from the project.
